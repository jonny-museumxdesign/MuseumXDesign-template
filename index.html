Resources for students learning HTML and CSS at the British Museum Samsung Centre

I had a great time coding with you guys. I hope you enjoyed it too and will carry on customising your projects.

To help your memory, I have set out a step-by-step guide to GitHub and JSBin below. Just follow the steps and you should be OK.

How to edit your code...

  1. Go to Thimble and login
  2. Make all the cusomisations you need to
  3. If you forgot how to write HTML and CSS you can either go back to the presentation or google it!
  4. Once you have made all of your changes you now need to reupload your project to Github

Save your edits in GitHub...

  1. In your Thimble go to your index.html sand click somewhere in the code
  2. Hit Ctrl+A to select all the code, then Ctrl+C to copy it
  3. Go to your GitHub tab, click on Code
  4. Click on the index.html file
  5. Click on the little pencil icon in the top right hand corner
  6. Click somewhere in the code, then hit Ctrl+A to select all the code
  7. Delete it!
  8. Hit Ctrl+V to paste your edited HTML
  9. Scroll down and press the green Commit changes button
  10. Go back to your Thimble and go to your custom-styles.css file and click somewhere in the code
  11. Hit Ctrl+A to select all the code, then Ctrl+C to copy it
  12. Go to your GitHub tab, click on Code
  13. Click on the stylesheets folder, then on the custom-styles.css file
  14. Click on the little pencil icon in the top right hand corner
  15. Click somewhere in the code, then hit Ctrl+A to select all the code
  16. Delete it!
  17. Hit Ctrl+V to paste your edited CSS
  18. Scroll down and press the green Commit changes button

Check your new page...

In your GitHub repository, click on Settings

Scroll down to the GitHub Pages box and click on the url.

You should see your new page :)

Keep learning...

  • CSS Diner is really great for getting your head around tricky CSS selectors. It's also quite fun.
  • Flexbox Froggy For learning how to position elements in CSS
